## 📝 SysReptor Setup

📺 Video walkthrough:  
https://youtu.be/dT7XgN8uuSU

To install and run the setup script in one command:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/strikoder/OSCP-Prep/main/Report/sysreptor_setup.sh)
```

> **Note:** The watermark may appear at the front of code blocks, which can affect reproducibility when submitting the report.  
> To remove it, go to:
>
> `Design → OSCP → PDF Designer`
>
> Then delete:
>
> ```html
> <div v-if="report.watermark" id="watermark-osid">{{ report.osid }}</div>
> ```
